# ゲーム のタイトル
* コインゲッター
## 実行環境の必要条件
* python >= 3.10.9
* pygame >= 2.1

## ゲームの概要
* 上から落ちてくるコインを集めてスコアを稼ぐ

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
### 担当追加機能
* ほげほげ
* ふがふが
* ぴよぴよ
* Time機能（担当：マナト）：時間の管理を行うクラスの追加、Timeが0になったら強制終了する機能
